# Підключення до Prom.ua — покрокова інструкція

---

## Що це і навіщо

Після підключення ваш магазин буде автоматично:
- **Отримувати замовлення** з Prom.ua прямо в адмінпанель
- **Оновлювати ціни та залишки** товарів на Prom.ua коли ви їх змінюєте
- **Передавати статуси** замовлень назад у Prom (підтверджено, відправлено тощо)
- **Передавати ТТН** Нової Пошти автоматично

---

## Крок 1 — Отримайте API ключ у кабінеті Prom.ua

1. Відкрийте my.prom.ua і увійдіть у свій акаунт
2. У лівому меню знайдіть **Налаштування → API**
3. Натисніть кнопку **Згенерувати токен**
4. Скопіюйте ключ — він виглядає приблизно так:
   `a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4e5f6a1b2`

> ⚠️ Збережіть ключ одразу — після закриття сторінки він більше не відображається

---

## Крок 2 — Налаштуйте інтеграцію в адмінпанелі

1. Відкрийте **Налаштування магазину → вкладка "Інтеграції"**
2. Увімкніть перемикач **"Інтеграція з Prom.ua"**
3. Вставте API ключ у поле **"Ваш API ключ"**
4. Натисніть кнопку **"Перевірити зв'язок"**
   - ✅ **"З'єднання успішне"** — все гаразд, йдемо далі
   - ❌ **"Неправильний API ключ"** — перевірте чи скопіювали ключ повністю, без зайвих пробілів
5. Натисніть **"Зберегти налаштування"**

---

## Крок 3 — Налаштуйте отримання замовлень (Вебхук)

Вебхук — це адреса вашого сайту, на яку Prom.ua надсилатиме нові замовлення автоматично.

1. На вкладці "Інтеграції" знайдіть поле **"URL вебхука"** і скопіюйте його кнопкою копіювання:
   `https://ваш-сайт.ua/prom/webhook`
2. Поверніться до кабінету Prom.ua
3. Перейдіть **Налаштування → API → Вебхуки**
4. Натисніть **"Додати вебхук"**
5. Вставте скопійований URL
6. Подія: оберіть **"Нове замовлення"**
7. Збережіть

> ✅ Тепер кожне нове замовлення з Prom.ua одразу з'явиться у вашій адмінпанелі в розділі "Замовлення"

---

## Крок 4 — Завантажте товари на Prom.ua

Є два способи — оберіть один:

### Спосіб А: XML-фід (рекомендовано для великих каталогів)

Підходить якщо у вас **більше 100 товарів**. Prom сам завантажує ваш каталог раз на добу.

1. На вкладці "Інтеграції" оберіть **"Підхід А: XML/YML фід"**
2. Натисніть **"Згенерувати XML зараз"** — файл створиться на сервері
3. Скопіюйте **"URL вашого XML-фіду"**:
   `https://ваш-сайт.ua/prom/feed.xml`
4. У кабінеті Prom.ua перейдіть **Товари → Імпорт товарів → Імпорт з YML**
5. Вставте URL фіду та збережіть
6. Prom завантажить ваші товари (може зайняти до кількох хвилин)

> 💡 Після кожного великого оновлення каталогу натискайте "Згенерувати XML зараз"

### Спосіб Б: API (для миттєвого оновлення цін і залишків)

Підходить якщо товари вже є на Prom і вам потрібно **синхронізувати ціни та залишки в реальному часі**.

> ⚠️ Для цього способу потрібно щоб у полі `prom_product_id` у вашій БД були записані ID товарів з Prom (дивіться Крок 5)

1. На вкладці "Інтеграції" оберіть **"Підхід Б: API (миттєво)"**
2. Натисніть **"Додати всі товари в чергу"**
3. Натисніть **"Обробити чергу (50 шт.)"** — відправить перші 50 товарів на Prom
4. Повторюйте натискання поки лічильник **"В черзі"** не стане 0

> 💡 Надалі система автоматично додає товар у чергу щоразу коли ви змінюєте його ціну або залишок

---

## Крок 5 — Прив'язка ID товарів (тільки для Способу Б)

Щоб система знала який ваш товар відповідає якому товару на Prom — потрібно заповнити поле `prom_product_id`.

Як дізнатись ID товару на Prom:
1. У кабінеті Prom.ua відкрийте потрібний товар
2. Подивіться на адресний рядок браузера:
   `my.prom.ua/cms/products/123456789/edit`
3. Число `123456789` — це і є `prom_product_id`

Заповнити можна через phpMyAdmin:
```sql
UPDATE products SET prom_product_id = 123456789 WHERE id = 15;
```
де `123456789` — ID товару на Prom, `15` — ID товару у вашій БД.

## Що відбувається після підключення автоматично

| Подія | Що станеться |
|---|---|
| Нове замовлення на Prom | З'явиться у розділі "Замовлення" адмінпанелі |
| Ви змінили ціну товару | Товар потрапляє в чергу на оновлення в Prom |
| Ви змінили залишок | Товар потрапляє в чергу на оновлення в Prom |
| Ви змінили статус замовлення | Статус оновлюється на Prom автоматично |
| Ви поставили статус "Відправлено" і вказали ТТН | ТТН передається в Prom автоматично |

## Де дивитись помилки

Якщо щось пішло не так — відкрийте файл логів на сервері:
```
storage/logs/prom.log
```

Приклади записів у лозі:
```
[2025-01-15 14:32:01] [OK] GET /orders/status_list | HTTP:200
[2025-01-15 14:33:05] [WEBHOOK_OK] Замовлення Prom #987654321 → order #142, товарів: 3
[2025-01-15 14:35:12] [API_ERROR] POST /products/edit | HTTP:401: Invalid token
```

Якщо бачите `401 Invalid token` — значить змінився API ключ, оновіть його на вкладці "Інтеграції".

## Часті питання

**Замовлення з Prom не приходять**
- Перевірте що URL вебхука правильно вказано в кабінеті Prom
- Перевірте що сайт доступний з інтернету (не localhost і не 127.0.0.1)
- Перевірте файл `storage/logs/prom.log`

**Ціни не оновлюються на Prom**
- Перевірте що поле `prom_product_id` заповнене для товару
- Відкрийте вкладку "Інтеграції" і натисніть "Обробити чергу"
- Перевірте лог на наявність помилок

**Хочу тимчасово вимкнути інтеграцію**
- Вкладка "Інтеграції" → вимкніть перемикач → "Зберегти"
- Вебхуки від Prom будуть ігноруватись, черга зупиниться

**Змінився API ключ**
- Згенеруйте новий ключ у кабінеті Prom.ua
- Вставте його на вкладці "Інтеграції" → натисніть "Перевірити зв'язок" → "Зберегти"

---

*Якщо щось незрозуміло — перевірте файл `storage/logs/prom.log`,
там завжди є детальна інформація про те що відбувається між вашим сайтом і Prom.ua.*
